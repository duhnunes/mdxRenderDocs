Para começar, você só precisa criar uma pasta com um arquivo `.mdx` dentro de `app/docs`.  
Automáticamente a rota/link/página será gerada na sidebar.  
Você pode modificar a ordem dos links dentro da categoria e também a ordem da categoria, utilizando `frontmatters` no `.mdx` e usando `.json` na pasta.

- Ordenando os links dentro da categoria:
  No topo do arquivo, antes de tudo, você deve colocar o frontmatters:
  ```markdown
  ---
  title: Título do link na sidebar    // Título que aparece na sidebar
  description: Descrição na página    // descrição que aparece no componente acima do conteúdo
  link_position: 1    // número da posição desejada
  ---
  ```
- Ordenando as categorias:
  Adicione um arquivo com nome `_category.json` dentro da pasta:
  ```json
    {
      "title": "Titulo da categoria", // Título que aparece na sidebar
      "sidebar_position": 1 // número da posição que você quer na ordem
    }
  ```
